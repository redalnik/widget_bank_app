# Учебный проект на Python: "Виджет для банка".
## Описание проекта
Это виджет, который будет показывать несколько последних успешных банковских операций клиента.
## Цель проекта
- Закрепить полученные знания в написании кода.
- Повышать навыки программирования.
- Освоить работу в Git (продвинутый пользователь).
- Согласно инструкциям и рекомендациям к концу курса написать работающей виджет для банковского приложения.
## Структура проекта
* src\mask.py - модуль, с функциями для маскировки номера карты или счета.
* src\processing.py - модуль, с функциями для обработки банковских операций.
* src\widget.py - модуль, с функцией форматирования даты и объединением функций маскировок.
## Установка
``` poetry install ```
## Пример использования
Для того, что бы запустить текущие возможности написанных функций,
нужно сначала установить зависимости.
Затем можно запустить файл main.py, следующим образом:
``` python main.py ```
