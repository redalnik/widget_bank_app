# Учебный проект на Python: "Виджет для банка"

## Описание проекта

Это виджет, который будет показывать несколько последних успешных банковских операций клиента.  
Цель — создать простое приложение, демонстрирующее работу с банковскими транзакциями из JSON, CSV и XLSX файлов.

## Цель проекта

- Закрепить полученные знания в написании кода.
- Повысить навыки программирования на Python.
- Научиться работать с Git на продвинутом уровне.
- Создать работающий виджет для банковского приложения согласно инструкциям курса.

## Структура проекта

```
widget_bank_app/
├── main.py                      # Основная логика программы
├── src/                         # Исходный код приложения
│   ├── mask.py                  # Маскировка номеров карт и счетов
│   ├── generators.py            # Генераторы транзакций
│   ├── processing.py            # Обработка транзакций
│   ├── widget.py                # Форматирование даты и маскировок
│   ├── decorators.py            # Декораторы (логирование)
│   ├── external_api.py          # Взаимодействие с внешним API
│   ├── utils.py                 # Работа с JSON-файлами
│   └── csv_excel_reader.py      # Чтение CSV/XLSX файлов
├── tests/                       # Тесты
│   ├── test_masks.py
│   ├── test_generators.py
│   ├── test_decorators.py
│   ├── test_external_api.py
│   ├── test_csv_excel_reader.py
│   ├── test_utils.py
│   ├── test_processing.py
│   └── test_widget.py
├── .env.template                # Шаблон переменных окружения
└── README.md                    # Документация
```

## Установка

Клонируйте репозиторий:
Выполнить команду ```git clone``` + (https://github.com/redalnik/widget_bank_app.git)

**Установите зависимости через Poetry:**
``` poetry install ```

**Заполните переменные окружения:**
.env.template -> .env

## Пример использования

Чтобы запустить программу:

1. Установите зависимости.
2. Заполните `.env` файл.
3. Запустите скрипт: main.py

Пример взаимодействия:
* Привет! Добро пожаловать в программу работы с банковскими транзакциями. 
* Выберите необходимый пункт меню: 

1. Получить информацию о транзакциях из JSON-файла
2. Получить информацию о транзакциях из CSV-файла
3. Получить информацию о транзакциях из XLSX-файла
